# Política de Privacidade – Calendário Inteligente

Este aplicativo respeita a privacidade dos usuários.

## Coleta de dados
O Calendário Inteligente não coleta, armazena, transmite ou compartilha dados pessoais.

## Armazenamento
Todas as informações inseridas pelo usuário permanecem armazenadas localmente no dispositivo.

## Compartilhamento
Nenhuma informação é compartilhada com terceiros.

## Permissões
O aplicativo pode solicitar permissões necessárias apenas para funcionamento básico, como notificações locais.

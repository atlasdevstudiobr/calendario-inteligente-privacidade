Política de Privacidade – Meu Peso: Lembretes e Evolução

Esta Política de Privacidade descreve como o aplicativo "Meu Peso: Lembretes e Evolução", desenvolvido por Atlas Dev, coleta, utiliza e protege as informações dos usuários.

1. Coleta de Informações
   O aplicativo não coleta, armazena ou compartilha dados pessoais identificáveis dos usuários. Todas as informações inseridas são armazenadas exclusivamente no dispositivo do usuário.

2. Uso das Informações
   As informações inseridas no aplicativo são utilizadas apenas para fornecer funcionalidades como acompanhamento de peso, lembretes e registros pessoais.

3. Armazenamento
   Todos os dados permanecem armazenados localmente no dispositivo do usuário. O aplicativo não envia dados para servidores externos.

4. Compartilhamento de Dados
   O aplicativo não compartilha dados com terceiros.

5. Permissões
   O aplicativo pode solicitar permissões necessárias para funcionamento adequado, como notificações. Nenhuma permissão é utilizada para coleta de dados pessoais sem consentimento.

6. Segurança
   Como os dados são armazenados localmente no dispositivo, o usuário é responsável pela segurança do seu aparelho.

7. Direitos do Usuário
   O usuário pode, a qualquer momento, excluir seus dados diretamente no aplicativo ou desinstalando o app.

8. Alterações nesta Política
   Esta Política de Privacidade pode ser atualizada periodicamente. Recomendamos que o usuário revise esta página regularmente.
